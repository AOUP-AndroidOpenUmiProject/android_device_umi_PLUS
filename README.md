UMi Plus - CyanogenMod 13.0/AOSP
==============

This branch is device for building of the CyanogenMod 13.0 and Android Marshmallow 6.X AOSP ROMs.

---

# About Device

![UMi Plus](http://schede.androidworld.it/images/umi_plus.png)

The UMi Plus (codenamed _"PLUS"_) is a flagship smartphone from UMi. The PLUS now supports CyanogenMod 13.0!

### Specifications

Component Type | Details
-------:|:-------------------------
CPU     | 2.0GHz Octa-Core MT6755 (Helio P10)
GPU     | Mali-T860
Memory  | 4GB RAM
Shipped Android Version | 6.0
Storage | 32GB
Battery | 4000 mAh
Display | 5.5" 1920 x 1080 px DPI 480
Rear Camera | 13MP (Samsung S5K3L8)
Front Camera | 5MP (GalaxyCore GC5005)
